# CS348_Task_Due
Task Due is a Task tracking program created for CS34800, a database course. Using a React/TypeScript frontend, an Express.js backend, and a MySQL database, the application allows users to create, view, update, delete, and filter tasks.

It also Includes dynamic UI components created from database data, utilization of indexes to speed up when finding a specific tasks with certain fields, and carefully designed queries using parameterized statements to protect against SQL injection attacks.
